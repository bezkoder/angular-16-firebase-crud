# Angular 16 Firebase CRUD example with Realtime DataBase | AngularFireDatabase

Build Angular 16 CRUD example with Firebase Realtime Database that uses `AngularFireDatabase` service.

- Each Tutorial has key, title, description, published status.
- We can make CRUD operations: create, retrieve, update, delete Tutorials.
